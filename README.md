# 📊 Week 12 CFAT Dashboard

## Overview
This repository hosts the **Week 12 CFAT Annotations Ops Plan Dashboard**, a live-updatable dashboard tracking key metrics for CFAT annotation operations. Built with HTML/CSS/JS and deployed via GitHub Pages for real-time updates.

## Live Demo
🔗 [View the live dashboard](https://sffeldtm.github.io/week12-cfat-dashboard/)  
*(Automatically updates when you push changes to this repo!)*

## Key Metrics Tracked
- **QBR Hallucinations**: Annotations remaining, adjudication status, HC, AHT  
- **GRS**: Language-specific progress (PT, FR, DE, IT, ES, JP, etc.)  
- **Videos in 30Secs**: Task counts and AHT  
- **Mission Bundles**: Progress toward total bundles  
- **Review Sharing Test**: Task status and HC  

## How to Update the Dashboard
1. **Edit `index.html`** (or your dashboard file) in this repo.  
2. **Push changes** to the `main` branch.  
3. GitHub Pages **auto-redeploys** within seconds!  

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript  
- **Hosting**: GitHub Pages (free, no setup required)
